etiqueta para la incercion de tablas

la etiqueta table se utiliza para poder incertar registros dentro de una tabla.
La etiqueta principal es " table " para decir al navegador que insertaremos una tabla
La etiqueta "tr" noos sirve para especificar que vamos a trabajar en una fila tr =fila
La etiqueta para decir que vamos a trabajar en una columna es "td" td=columna Ejemplo


La para colocar un borde en nuestra tabla debemos agregar un atributo "Border" a si tambien podemos agregar un padding a nuestra tabla con "cellpadding"
